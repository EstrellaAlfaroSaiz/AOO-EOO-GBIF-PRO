about=LRN/GBIF is a QGIS plugin for preliminary Red List spatial assessment workflows. It downloads GBIF occurrence records with methodological filters, supports manual points and imported point layers, calculates Area of Occupancy (AOO) using a configurable grid and Extent of Occurrence (EOO) using a convex hull, and exports IUCN-compatible Point Distribution data as CSV or shapefile. It includes audit tables for records removed by filters and tools to request an official GBIF occurrence download DOI for the GBIF records used. The results are intended to support expert review and do not replace a full IUCN Red List assessment.
